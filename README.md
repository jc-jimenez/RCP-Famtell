# RCP.ai

Proyecto base para la plataforma SaaS de diagnóstico empresarial con IA.

## Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- Supabase
- Prisma
- Stripe
- Anthropic Claude API
- Resend
- Twilio

## Estructura

- `src/app` — rutas App Router
- `src/components` — componentes UI
- `src/lib` — clientes de integración y utilidades
- `src/types` — tipos compartidos
- `prisma/schema.prisma` — esquema de datos

## Notas

La instalación de dependencias depende de tener Node.js y npm/pnpm/yarn disponibles en el entorno.
